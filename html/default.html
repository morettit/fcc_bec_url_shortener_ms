FreeCodeCamp API Basejump: Url Shortener Microservice

User stories:
    1) I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
    2) If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
    3) When I visit that shortened URL, it will redirect me to my original link.

Example usage - Short Url Creation:

https://fcc-bec-url-shortener-ms.herokuapp.com/new/https://google.com

Output:

{ "original_url": https://google.com, "shortened_url": http://fcc-bec-url-shortener-ms-morettit.c9users.io/575ac9febf11874613e94771 }

Example usage - Short Url Creation with Broken Link:

https://fcc-bec-url-shortener-ms.herokuapp.com/new/https://google.com/aaa

Output:

{"error": Link is broken.}

Example usage - Short Url Creation with NOT URL element:

https://fcc-bec-url-shortener-ms.herokuapp.com/new/thisisasimplestring

Output:

{"error": Wrong url format.}

Example usage - Retrieve URL:

https://fcc-bec-url-shortener-ms.herokuapp.com/575e8ef4ca49f4e81014c30d

Output:

Redirect to : http://www.twitter.com
